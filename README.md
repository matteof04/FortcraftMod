# FortcraftMod

This is a Minecraft mod for recreate Fortnite in Minecraft 1.12.2.

I've also created a Minecraft Sponge plugin, check it out!

I use this mod inside this modpack:

* Xaero's Minimap
* Optifine
* Flan's Mod
* Open Glider Mod
* Offline skins
* HideNames
* Grapple Mod
* Dynamic Sourrandings
